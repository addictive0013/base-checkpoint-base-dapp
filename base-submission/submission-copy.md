# Base Checkpoint

## App Name

Base Checkpoint

## Tagline

Ship one step

## Description

Log one daily checkpoint on Base and keep a clean streak board that shows what you shipped and when.

Character count: 98

## Category

Progress / streak / onchain utility

## Short Pitch

Base Checkpoint gives people a clean way to log one concrete daily move on Base and keep a readable public board of progress.

## Production Environment Variables

- `NEXT_PUBLIC_CHECKPOINT_CONTRACT_ADDRESS`: deployed checkpoint contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
