# Base Checkpoint

Dear reviewer,

This is a small Base project about base checkpoint. It lets a visitor making the main app action and leaves a record behind. The project is not trying to prove ownership through inboxes; it proves it with the same public pieces a user can open and verify.

The Base Dashboard registration is present under Build ID `6a04a5ee2be96789d34cf06c`. The public builder wallet is `0x954815bf0262b9cEE18C354BAe59c12E31f5F0Fa`. Transaction attribution uses Builder Code `bc_pxddh8bb`. The live deployment is https://base-checkpoint.vercel.app, and the source repository is https://github.com/addictive0013/base-checkpoint-base-dapp.

If you want to run it locally:

```bash
npm install
npm run dev
```

The working parts are simple: Next.js, React, TypeScript, Tailwind CSS, wagmi, viem, Base, Vercel. Local secrets stay local; `.env.example` is the only environment file meant for Git.
